Пазлы Инициализации

Эти пазлы находятся во вкладке init и интерпретируются перед инициализацией приложения Verge3D.

Содержание

  • Перечень Пазлов
    • configure application
    • setup preloader + percentage
    • load font

#

Перечень Пазлов

#

configure application

Позволяет задать параметры инициализации приложения, включая некоторые параметры создания контекста WebGL.

compressed assets
заставляет приложение загружать сжатую сцену в формате .xz вместо обычного .gltf (см. Asset Compression для большей информации).
default fullscreen button
включает стандартную кнопку полноэкранного режима, расположенную в правом верхнем углу.
transparent background
делает фон прозрачным, чтобы основные части веб-страницы были видны через canvas WebGL. Включите этот параметр, чтобы встроить 3D-контент на веб-страницу или использовать функцию дополненной реальности.
enable screenshots
позволяет правильно снимать скриншоты с WebGL canvas (устанавливает свойство preserveDrawingBuffer контекста WebGL в true).
fade annotations
делает аннотации тусклыми, когда они блокируются объектами сцены.

#

setup preloader + percentage

Удаляет прелоадер по умолчанию и передает обратные вызовы событий для обработки событий прогресса прелоадера.

В следующем примере пользовательский прелоадер представлен с контейнером HTML-элементом (например, <div>) с id "preloader". Внутри него находится другой элемент с id "loading progress" (например, <span>), в котором выводится % загрузки с помощью пазла persentage. Кроме того, ширина третьего элемента (например, <div>) динамически изменяется для графической визуализации прогресса загрузки.

#

load font

Загружает шрифт из указанного URL и делает его доступным через CSS-правило font-family.

Предварительные условия

Maya

Для начала необходимо установить Autodesk Maya 2018-2022, так как Verge3D работает поверх него. В случае отсутсвия программы, вы можете загрузить бесплатный пробный пакет с сайта Autodesk, или попробовать Verge3D для Blender.

Мы рекомендуем использовать последнюю стабильную версию Maya (2022 на момент написания статьи) для достижения наилучших результатов.

Операционная система

Мы рекомендуем использовать Verge3D на Windows 10, macOS 10.14/10.15 или последней версии RedHat/CentOS/Fedora Linux. Убедитесь, что у вас установлены последние версии видеодрайверов. Для установки не требуются права администратора.

Веб-браузер

Verge3D может работать в любом браузере, таком как Chrome, Firefox, Opera, Edge или Internet Explorer 11. Однако мы рекомендуем использовать браузер Chrome для разработки, так как он быстрый и предоставляет хорошие инструменты для разработки. Также вы можете сделать его браузером по умолчанию для более удобного предварительного просмотра.

Установка Verge3D на Windows

Скачайте пакет Verge3D и дважды щелкните на файле .exe, чтобы запустить программу установки.

Выберите папку для установки, которая имеет достаточно прав на чтение и запись (или лучше оставьте ее по умолчанию). Не рекомендуется устанавливать в системные папки, такие как "Program Files" и "Desktop".

После завершения установки вы можете открыть Maya. Если она была запущена, не забудьте закрыть и открыть ее снова. Теперь перейдите к настройкам Plug-in Manager...

...и активируйте плагин verge3d.py, включив флажки Loaded и Auto load.

Дополнительно вы можете активировать плагин gltfExport.py, если хотите экспортироватm сцены Verge3D через опцию меню File->Export All.... Включение gltfExport.py добавляет опцию типа файла Verge3D GLTF для диалогового окна Export All.

Теперь вы должны увидеть меню Verge3D в строке главного меню.

Также теперь у вас есть возможность запускать Verge3D's App Manager независимо от Maya, нажав на значок на рабочем столе или используя меню "Пуск".

Установка Verge3D на macOS

Скачайте пакет Verge3D XZ и распакуйте его в выбранный вами каталог. Убедитесь, что у вас достаточно прав для чтения/записи в этой папке. Установка в системный каталог не рекомендуется.

Если вы используете Maya версии до 2022 года, вам необходимо выполнить дополнительные шаги для установки Python (который входит в состав Command Line Tools). Если вы используете Maya 2022 или новее, перейдите к шагу с вызовом Plug-in Manager.

Для установки Command Line Tools откройте macOS Terminal и выполните команду:

xcode-select --install

Если инструменты не установлены в вашей системе, сценарий вызовет следующее диалоговое окно установки:

Дождитесь завершения установки, прежде чем приступать к дальнейшим действиям.

Теперь откройте Maya и перейдите к настройкам Plug-in Manager...

Нажмите на кнопку Browse

затем перейдите в директорию Path_To_Verge3D_Installation/maya_plugin/setup и откройте скрипт install.py.

В более новых версиях Maya может появиться следующее диалоговое окно - нажмите Allow.

Перезапустите Maya, снова откройте настройки Plug-in Manager и активируйте плагин verge3d.py, включив флажки Loaded и Auto load.

Если хотите экспортировать сцены Verge3D через опцию меню File->Export All... вам необходимо активировать плагин gltfExport.py. Включение gltfExport.py добавляет опцию типа файла Verge3D GLTF для диалогового окна Export All.

Теперь вы должны увидеть меню Verge3D в строке главного меню.

Настройка пути плагина вручную

Иногда меню Verge3D не отображается. Это может произойти по нескольким причинам:

  • Вы отключили опцию Auto-Configure в программе установки.
  • Вы установили и запустили неподдерживаемую версию Maya.
  • Вы сменили пользователя Windows.

В этом случае вы можете использовать следующую процедуру для настройки пути к плагину вручную.

Найдите файл с именем Maya.env, который обычно находится в следующих папках:

  • Windows - \Users\user\Documents\maya\VERSION
  • macOS - $HOME/Library/Preferences/Autodesk/maya/VERSION
  • Linux - $HOME/maya/VERSION

Отредактируйте этот файл, чтобы указать переменную окружения MAYA_PLUGIN_PATH следующим образом:

MAYA_PLUG_IN_PATH=\Users\user\verge3d_maya\maya_plugin

Сохраните файл и откройте Maya. Если она была запущена, не забудьте закрыть и открыть ее снова. Найдите плагин Verge3D в диалоге настроек Plug-In Manager. Активируйте его, нажав на флажки Loaded и Auto load. Теперь вы должны увидеть меню Verge3D в строке главного меню.

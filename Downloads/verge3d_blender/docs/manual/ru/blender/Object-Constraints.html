#

Ограничения и Настройки

Ниже перечислены объектные ограничения Blender, которые могут быть использованы в Verge3D. Параметры поддерживаемые Verge3D выделены зеленым цветом.

#

Copy Location

Данное ограничение заставляет объект, к которому оно применено, иметь такое же местоположение, как у его цели.

#

Copy Rotation

Данное ограничение заставляет объект, к которому оно применено, иметь такой же поворот, как у его цели.

#

Copy Scale

Данное ограничение заставляет объект, к которому оно применено, иметь такой же масштаб, как у его цели.

#

Copy Transforms

Данное ограничение заставляет объект, к которому оно применено, трансформироваться также, как и его цель.

#

Limit Location

Данное ограничение контролирует величину разрешённых перемещений вдоль каждой оси посредством минимального и максимального порогового значения.

#

Limit Rotation

Данное ограничение контролирует величину разрешённых поворотов вокруг каждой оси посредством минимальных и максимальных пороговых значений.

#

Limit Scale

Данное ограничение контролирует величину разрешённого масштабирования вдоль каждой оси посредством минимальных и максимальных пороговых значений.

#

Track To

Данное ограничение поворачивает объект в направлении цели, чтобы он всегда смотрел в сторону оси назначенной для To, в то время как другая ось Up постоянно поддерживалась максимально параллельно глобальной оси Z. Таким образом, благодаря данному ограничению вы можете реализовать псевдотрёхмерность.

#

Locked Track

Данное ограничение работает, как ограничение Track To, но с фиксированной осью.

#

Child Of

Данное ограничение является вариантом стандартных взаимоотношений типа родитель/потомок между объектами.

#

Floor

Данное ограничение позовляет использовать цель в качестве плоскости, которую объект не сможет пересечь.

#

Используйте с Пазлами

Все назначенные в Blender ограничения, после экспортирования могут быть удалены, отключены и включены с помощью пазлов. В отличие от Blender, убранные или отключённые ограничения не сбрасывают положение объекта, данный пазл оставит объект в том же положении, в котором он был до удаления или отключения ограничений.

Кроме того, ограничения Limit/Copy Location/Rotation/Scale могут быть назначены во время запуска с данным пазлами.

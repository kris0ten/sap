使用材质库 / Maya

Verge3D材质库是一个随时可用的材质合集，可以在Maya中将其指定给模型。下面是关于如何高效地安装和使用这些材质的推荐工作流程的描述。

  • 安装
  • 向项目添加材质
  • 设置环境
  • UV映射
  • 将纹理转移到到应用文件夹

#

安装

材质库在Soft8Soft的商店中以主题包的形式提供，见 Maya Essential Material Pack。

只需将材质库的压缩包解压缩到计算机上的任何文件夹即可。在这个文件夹里有mb文件（每个材质一个文件）。材质中使用的纹理位于material_maps文件夹中。文件夹Environments包含可用于环境的3个HDR图像变体（每个变体有3种分辨率：1K、2K和4K)。

#

向项目添加材质

要在项目中使用这些材质，请在Hypershade编辑器窗口中使用 文件 -> 导入方式。

导航到库文件夹，选择mb文件，在“加载选项”中指定“不加载引用”，然后点击“导入”。

材重要被导入后，您即可将其指定给模型了。选择一个模型，鼠标右键单击选择 指定现有材质 -> 材质名称。

最后一步是删除材质库中未使用的引用。

#

设置环境

基于Eevee的材质需要设置好环境，否则材质看起来很暗，且不反光。环境已经在Verge3D的默认立方体项目中设置好了，您可以把它作为自己应用的基础。

另外，您也可以从头开始配置环境，如本视频教程所示： 或者，您也可以从头配置环境，如下所示视频教程： YouTube： Verge3D for Blender Basics - Part 3 - Materials and Environment Bilibili: Verge3D for Blender基础 - 03.材质与环境 为此，您可以使用材质包的 environments 文件夹中的任一HDR纹理。

#

UV映射

材质库中的大多数材质都要求模型进行UV映射。请展开模型已取得最佳效果。

#

将纹理转移到到应用文件夹

为了您应用的完整性（如果您要将项目源转移给第三方，这一点很重要），您可以将被导入的材质所使用的纹理保存在应用文件夹中，如下所述。

  1. 打开Maya中的 “文件路径编辑器”。
  2. 找到已导入材质所使用的纹理。将这些纹理从material_maps复制到应用文件夹中。
  3. 在编辑器中选中这些材质，然后点击重新指定文件路径按钮。在出现的新对话框中，指定新路径到应用目录，启用“重新指定当前已解析文件的路径”选项，然后单击“重新指定路径”。

您可以通过将应用的文件夹移动到文件系统中的另一个位置来检查应用的完整性。打开 mb(或ma) 文件，确保没有产生缺少资产的错误。

动画约束 / 3ds Max

本页包含可以在Verge3D中使用的3ds Max动画 约束 的有关信息。

  • 约束和设置
    • 位置
    • 方向
    • 注视
  • 与拼图一起使用

#

约束和设置

下面列出了可与Verge3D一起使用的3ds Max的动画约束。Verge3D所支持的参数被绿色覆盖。

#

位置

此约束强制设置被所应用的对象和目标的位置相匹配。

参见Autodesk网站上关于位置约束图的更多信息： 帮助页面。

#

方向

此约束强制设置被应用的对象和目标的旋转相匹配。

参见Autodesk网站上关于方向约束的更多信息： 帮助页面。

#

注视

这个约束将旋转应用于他的所有者，因此它可以始终看向目标。因此，你可以用这个约束条件来实现广告牌效果。

参见Autodesk网站上关于方向约束的更多信息： 帮助页面。

#

与拼图一起使用

所有的约束，一旦在3ds Max中指定并导出，就可以用拼图的方式删除、静默或解除静默。与3ds Max不同的是，移除/静默约束并不会重置对象——这个拼图只是让对象在删除或静默之前具有相同的变换。

此外， 位置和方向约束也可以通过拼图在运行时中指定。

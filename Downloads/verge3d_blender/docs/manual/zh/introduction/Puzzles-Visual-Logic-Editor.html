可视化逻辑编辑器，也称为拼图编辑器

Verge3D拼图编辑器是一个用于创建行为场景的工具，功能强大且容易上手。借助拼图，您可以为3D内容添加与用户之间的交互式响应。

如果您不擅长代码，这个工具将会显著提升您创作的能力。如果您本身就是经验丰富的程序员，您也会发现其在原型设计与生产中的好用之处。

#

用户界面

布局

拼图可以从左侧扩展工具箱中推拽到主工作区。三维渲染预览视窗浮动显示在右上角，此窗口可以移动、调整大小、隐藏、暂停及检查帧速率（FPS）。

控制 按钮位于页面的底部。选项卡显示在页面顶部，默认选项卡为为 init（初始化工作区） 和 main（主工作区）。

每个工程的布局设定（选项卡、视窗位置和大小）都是单独保存的，所以当您使用另一个浏览器或者另一台工作中打开拼图，所有设置都可恢复。

视窗大小调整与重置

拖拽视窗的边角以调整大小。

您可以通过右键点击工作区的空白处，选择菜单中的Reset Viewport Dimensions来重置视窗的位置和大小：

刷新编辑器

点击浏览器的标准刷新页面按钮，即可重新加载编辑器与运行在视窗中的应用。

所有未保存的拼图都将被移除，所有的变量和场景对象属性都将重置为初始值。因此，您可以在开发迭代中使用此方法来清除应用状态。

#

控件

运行

点击此按钮后，编辑器内的拼图即会生效。点击运行按钮后，由引擎解释当前存在于工作区所有选项卡内的拼图，并生成JavaScript代码。然后，浏览器会执行代码并应用于后台运行的应用。

大多数情况下，您可以反复点击运行按钮来执行拼图中的调整。但是，如果要重置应用状态并执行初始化运行，则需要点击浏览器的刷新按钮。刷新页面后，拼图自动生效。请确保在刷新前保存修改，否则将丢失为未保存的内容。

保存

击此按钮保存拼图。与运行操作类似，点击保存按钮后，引擎会解释当前工作区所有选项卡内的拼图，并生成JavaScript代码，但并不执行。所有的拼图和生成的代码都分别作为文件visual_logic.xml和visual_logic.js保存在应用的目录中。

保存操作需要本地开发服务器处在运行状态该服务器会在Blender，3ds Maya或Maya的第一个实例启动时自动启动。但如果服务器因为某种原因关闭，在会在尝试保存拼图时显示连接错误：

在这种情况下，您应该启动blender，3ds Max或Maya。如果这些软件已经在运行，请尝试再次启动应用管理器（App Mamager）。

另请参阅 备份.

#

选项卡

你可以在选项卡中重新排列拼图的顺序，以使其更易于管理和响应。要创建新的选项卡，请点击顶部栏中的“加号”按钮，并为此选项卡命名。

现在您可以在此选项卡内添加拼图了。如需移动一些拼图到另一个选项卡，请使用剪切 Ctrl-X 和 粘贴 Ctrl-V 快捷键。所有的选项卡都会共享同一个命名空间，所以您可以在一个选项卡中定义过程变量，在另一个选项卡中调用。

要删除某个选项卡，请点击选项卡名称右侧的叉号。请谨慎使用此操作，此操作无法撤销，除非备份或还原。被删除标签页内的所有拼图都会被清除。

从3.0版开始，不再有用于重命名选项卡的专用控件。但是，您依然可以通过在应用文件夹内编辑visual_logic.xml的方式来重命名选项卡。

重命名后，选项卡名称将在刷新编辑器后更新。

#

右键单击菜单（在拼图上）

可通过右键（在工具箱中或在工作区中）单击拼图调用上下文菜单。可快速调用许多非常有用的操作。

Duplicate

创建一组拼图的副本。您还可以使用“复制/粘贴”快捷键Ctrl-C/Ctrl-V执行同样的操作。

Add Comment / Remove Comment

为拼图添加注释。通过单击拼图左上角的问号按钮显示和关闭注释，点击后展开对应的注释文本窗口。注释可以在开启和关闭的状态下存储。

可以通过在右键菜单中选择“Remove Comment”来移除注释。

Inline Inputs / External Inputs

默认情况下，一个拼图可以是inlined（单行状态）或external（扩展列表状态）形态。此菜单选项用于切换两种形态。两者的区别只是表面上的，但有时候转换形态可以让拼图更紧凑及可读性更好。

Collapse / Expand

可以折叠一组拼图，以使其更紧凑。

Disable / Enable

使用此拼图可以对某段拼图跳过执行。

Delete

删除一组拼图。或者使用del键。

Save to Library

保存一组拼图到 库 ，以便在之后的工作中可以快速复用。

Help

打开对应的文档帮助页面，可在类目的工具箱中使用，

...也可在编辑器工作区使用。

#

右键单击菜单（在工作区上）

Undo / Redo

撤销或重做您最近的操作，也可以使用快捷键Ctrl-Z/Ctrl-Shift-Z。

Straighten Up

清理并重新放置当前选项卡内的所有拼图，使其排列为一条直线。

Collapse / Expand

工作方式与单组变量类似，不同之处在于它影响选项卡中的所有拼图。

Delete

工作方式与单一组变量类似，不同之处在于它删除了选项卡中的所有拼图。

Reset Viewport Dimensions

重置viewport窗口的位置和尺寸以更好地适应屏幕分辨率。

#

备份

每次单击Save按钮时，以前保存的文件visual_logic.xml都会复制到名为v3d_app_data/puzzles_backup的文件夹中，并在文件名中添加时间戳。

这允许您在需要时恢复先前保存的拼图修订版本。只需将文件复制回应用文件夹根目录，并从其名称中删除时间戳既可。刷新编辑器即可更新拼图。

#

插件 | Plugins

拼图编辑器支持插件系统。拼图插件可以理解为创建自定义的拼图模组。Verge3D发行版目录下包含一个案例插件， 电子商务插件 用于集成Verge3D应用和Woocommerce。但您也可以安装更多的插件，甚至创建自己的插件。在此查看更多信息：插件.

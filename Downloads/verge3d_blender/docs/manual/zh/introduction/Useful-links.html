第三方Verge3D插件

插件是扩展Verge3D核心功能的一种简洁和易用的方式。下面是由第三方专家与公司开发的Verge3D插件列表：

  • CSS3D — 用于在三维对象表面嵌入网站、视频或地图的插件。
  • Teleport — 在3D场景中添加可点击的导航点，用于移动第一人称摄像机。
  • 2D Canvas Editor by CyberFox — 功能强大的画布工具，可以直接在3D模型上绘图、书写和压印纹理。
  • 尺子 — 实现交互式标尺测量三维物体尺寸的插件。
  • Screen Recorder — 实时录屏插件。
  • Draggable Texture — 在模型上实时平移、旋转和缩放纹理。
  • Color Plugin — 为基于Verge3D的应用提供简易的颜色选择器。
  • Canvas — 使用拼图实时绘制基于矢量的纹理。
  • Virtual Joystick — 用于移动设备的虚拟操纵杆控制器。
  • 简单平面主题 — 拼图编辑器的平面主题。
  • GUI — 用于添加简单控制，实时调整场景图形的插件。
  • Save/Load URL — 用于保存和加载带有配置的URL的插件。

在 论坛 上查看和讨论更多关于插件开发的信息。

后期处理 | Post-processing

这些拼图用于设置多样化的后期处理效果。

内容

  • Live Preview(实时预览)
  • 拼图参考
    • bloom(辉光)
    • Bright/Contrast(亮度/对比度)
    • depth of field(景深)
    • grayscale(灰度)
    • ambient occlusion(环境光遮蔽)
    • simple refraction(简单折射)
    • screen space reflection/refraction(屏幕空间反射/折射)
    • remove post-processing effects(移除后期处理效果)

#

Live Preview(实时预览)

后期处理的拼图支持实时预览功能。启用实时预览功能后，您可以调整后期处理效果，并在拼图视窗中查看结果，无需运行或保存/重新加载拼图。

实时预览功能是默认启用的，并且支持在拼图没有连接输入时预览。

#

拼图参考

#

bloom(辉光)

启用或改变辉光效果的参数。

参数：

  • threshold(阈值) - 渲染辉光效果的最小照明强度。
  • strength(强度) - 辉光强度。
  • radius(半径) - 辉光的的模糊半径。

#

Bright/Contrast(亮度/对比度)

允许对场景渲染的亮度和对比度进行调整。

参数：

  • brightness(亮度) — 图像亮度系数。
  • contrast(对比度) — 图像对比度系数。

#

depth of field(景深)

启用或改变景深（DOF）效果的参数。

参数：

  • focus(对焦) - 景深对焦距离。
  • aperture(光圈) - 相机光圈系数。
  • maxblur(最大模糊) - 景深模糊半径。
  • depth leak threshold(深度泄露阈值) - 消除DOF距离伪影的阈值。

#

grayscale(灰度)

启用灰度效果。

#

ambient occlusion(环境光遮蔽)

启用或改变屏幕空间环境遮蔽（SSAO）效果的参数。

参数：

  • radius(半径) - SSAO效果的半径。
  • min distance(最小距离) - 距离阈值，用以减少因对象位置太近而造成的伪影。
  • max distance(最大距离) - 距离阈值，用以减少因对象位置太远而引起的伪影。

#

simple refraction(简单折射)

启用指定材质的简单折射效果。

参数：

  • material(材质) - 用于指定效果的材质。
  • distance(距离) - 折射光线的最大长度。
  • render after refraction(折射后渲染) - 从正常渲染队列中排除的对象，应用折射效果后再进行渲染。

#

screen space reflection/refraction(屏幕空间反射/折射)

启用屏幕空间反射/折射（SSR）效果。

参数：

  • for material(指定材质) - 用于指定该效果的材质。
  • steps(步数) - 沿着反射/折射光线的深度搜索步数（数值越高=质量越好，性能越差）。
  • stride(步幅) - 搜索步骤的步幅（以像素为单位）（数值越大=质量越查，性能越好，1像素=最高质量）。
  • Binary search steps(二进制搜索步数) - 二进制搜索迭代的数量（数值越大=质量越好，性能越差）。
  • thickness(厚度) - 消除SSR伪影的系数（仅用于折射）。
  • max distance(最大距离) - 反射/折射光线的最大长度。
  • resolution(分辨率) - 深度缓存的分辨率乘数（从0到1；1=最佳质量）。
  • jitter(抖动) - 改善深度搜索的干扰系数（从0到1）。
  • render after(在...后渲染) - 在应用SSR之后才被渲染的对象。

#

remove post-processing effects(移除后期处理效果)

移除全部后期处理效果。

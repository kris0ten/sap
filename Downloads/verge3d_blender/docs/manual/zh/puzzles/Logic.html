if / else

通过这些拼图可以在拼图中实现分支（控制流）。

假设，您的场景中有一个立方体和一个球体。当用户点击球体时将其隐藏，但前提是 用户之前点击过立方体。这样的逻辑很简单，可以使用一个 变量 来记住立方体是否被点击过。

如果需要 比较 一个有多个值的变量，则需要更多的逻辑分支。在这种情况下，您可以通过点击齿轮图标来扩展if拼图，拖拽添加更多的插槽：elseif 或 else。

true / false

逻辑拼图的操作只有两个值，true和false，也被称为布尔值。

这些值被 comparison(比较) 拼图，以及逻辑运算符 AND/OR 和 NOT作为输出返回，并被提供给分支拼图 if/else 和 test。

#

null

严格来说，null值与逻辑代数没有关系。如果一个变量被设置为null，意味着它没有被设置为true或false，而是被置于 "未决定" 状态。

您可以使用null来明确地初始化一个新创建的变量。

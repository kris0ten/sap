使用材质库/Blender

Verge3D材质库是一个随时可用的材质合集，可以在Blender中将其指定给模型。下面是关于如何高效地安装和使用这些材质的推荐工作流程的描述。

  • 安装
  • 向项目添加材质
  • 设置环境
  • UV映射
  • 配置材质
  • 图像分辨率
  • 将纹理转移到到应用文件夹

#

安装

材质库在Soft8Soft的商店中以主题包的形式提供，例如：

  • 基础材质包 （基于Eevee）。
  • 珠宝材质包 （基于Eevee）。
  • 汽车材质包（Eevee为基础，正在开发中）。

只需将材质库的压缩包解压缩到计算机上的任何文件夹即可。在这个文件夹里有blend文件（每个材质一个文件）。材质中使用的纹理位于material_maps文件夹中。文件夹Environments包含可用于环境的3个HDR图像变体（每个变体有3种分辨率：1K、2K和4K)。

#

向项目添加材质

要在项目中使用这些材质，请打开文件/附加(File/Append)对话框。

然后导航到材质库文件夹，点击打开带有要添加的材质的blend文件。

打开“Material”文件夹...

...选择材质，然后单击“追加”按钮。

材质被追加后，您即可将其指定给模型了。选择一个模型，转到 材质属性 选项卡，点击材质名称附近的小按钮（若还没有分配材质，则点击 新建 按钮），然后从列表中选择刚才附加的材质。

#

设置环境

基于Eevee的材质需要设置好环境，否则材质看起来很暗，且不反光。环境已经在Verge3D的默认立方体项目中设置好了，您可以把它作为自己应用的基础。另外，您也可以从头开始配置环境，如本视频教程所示： 或者，您也可以从头配置环境，如下所示视频教程：

  • YouTube: Verge3D for Blender Basics - Part 3 - Materials and Environment
  • Bilibili: Verge3D for Blender基础 - 03.材质与环境

    • 为此，您可以使用材质包的 environments 文件夹中的任一HDR纹理。

    #

    UV映射

    材质库中的大多数材质都要求模型进行UV映射。请展开模型已取得最佳效果。

    #

    配置材质

    大多数着色器提供了一组可调整的输入节点，您可以创建一个符合需要的独特材质。

    除此之外，还有一些特殊的输入节点，可以方便地连接烘焙好的法线贴图与AO贴图。

    #

    图像分辨率

    一些材质可以使用（最高至8k的）高分辨率纹理，但这会影响加载速度和性能表现。如果应用不需要这样的细节级别，请考虑重新缩放图像。

    #

    将纹理转移到到应用文件夹

    为了您应用的完整性（如果您要将项目源转移给第三方，这一点很重要），您可以将附加材质使用的纹理保存在应用文件夹中，如下所述。

    打开Blender中的 图像编辑器 （或 UV编辑器 )。从下拉菜单中选择一个纹理，并检查其文件路径。

    如果纹理看起来是从材质库文件夹中引用的，请使用 图像 / 另存 菜单重新保存到应用文件夹中。

    搜索材质库的其他纹理，重复此操作，直到所有的纹理都被重新定位。

    您可以通过将应用的文件夹移动到文件系统中的另一个位置来检查应用的完整性。打开 blend 文件并调用 文件 / 外部数据 / 报告缺失的文件 菜单。

    如果 blend 文件中使用的任何纹理有缺失，在视口底部会显示黄色高亮的警告信息。

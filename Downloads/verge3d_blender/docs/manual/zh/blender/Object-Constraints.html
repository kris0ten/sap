#

约束和设置

下面列出了可与Verge3D一起使用的Blender对象约束。Verge3D所支持的参数被绿色覆盖。

#

复制位置

此约束强制设置被所应用的对象和目标的位置相匹配。

#

复制旋转

此约束强制设置被应用的对象和目标的旋转相匹配。

#

复制缩放

此约束强制设置被应用的对象和目标的缩放相匹配。

#

复制变换

此约束强制设置被应用的对象和目标的变换相匹配。

#

限定位置

此约束通过下限和上限限制了沿每个轴的允许平移量。

#

限定旋转

此约束通过下限和上限限制了围绕每个轴的允许旋转量。

#

限制缩放

此约束通过下限和上限限制了围绕每个轴的允许缩放量。

#

跟随到(Trace To)

这个约束将旋转应用于它的所有者，因此它总是将一个给定的To轴指向它的目标，而另一个Up轴则永久保持与全局Z轴尽可能一致。因此，你可以用这个约束条件来实现广告牌效果。

#

锁定跟踪(Locked Track)

此约束的作用类似于Track To约束，但有一个锁定轴。

#

子级

这是对象之间的标准父/子关系的约束版本。

#

基面

此约束允许你将其目标设置为所有者无法到达的平面。

#

与拼图一起使用

所有的约束，一旦在Blender中指定并导出，就可以用拼图的方式删除、静默或解除静默。与Blender不同的是，移除/静默约束并不会重置对象——这个拼图只是让对象在删除或静默之前具有相同的变换。

此外， Limit/Copy Location/Rotation/Scale约束可以通过拼图在运行时中指定。

安装/Blender

  • 先决条件
  • 下载Verge3D
  • 在Windows中安装
  • 在macOS或Linux中安装

#

先决条件

Verge3D可以基于Blender运行，这是一个开源免费的3D建模和动画软件，请预先安装对应的版本（支持2.80-2.93版）。您可以从 Blender官方网站免费下载。

#

下载Verge3D

下载适用于您的操作系统的 Verge3D for Blender （Windows的.exe安装包，或macOS与Linux的XZ压缩包）。

#

在Windows中安装

双击 .exe 文件以运行安装程序。

选择一个有足够读写权限的安装目录（建议使用默认路径）。不建议在“Program Files”或“Desktop”等系统目录中安装。

安装完成后即可打开Blender。如果Blender正在运行，请先关闭然后再次打开。

打开“偏好设置”窗口并转到“插件”选项卡。找到并启用Verge3D插件，然后单击“保存用户设置”并关闭窗口。

从2.17版本开始，您也可以通过桌面或开始菜单中的Verge3D快捷方式图标来打开应用管理器，这种启动方式无需启动Blender。

#

在macOS或Linux中安装

将XZ压缩包解压到您选择的目录中。请确保您有足够的权限在此文件夹中进行读/写操作。不建议在系统目录中安装。

在Blender中，打开编辑菜单中的偏好设置窗口。

在文件路径选项卡中，找到脚本字段，并将其路径指向您安装Verge3D的文件夹。关闭“偏好设置”窗口并 重启Blender.

再次打开“偏好设置”窗口并转到“插件”选项卡。找到并启用Verge3D插件，单击“保存用户设置”后关闭窗口。

安装完成！
